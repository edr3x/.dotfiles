# Neovim Config

**Base config for Neovim**

## Requirement

### Packer

```sh
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```

### To copy form editor to clipboard

- To do this we need tool called `xclip`

```sh
sudo pacman -S xclip
```
